## 1. Продукт и контекст использования

**«Вертикаль»** — мобильное приложение для самостоятельной записи на тренировки по скалолазанию.

**Специфика среды (Environmental Context):**
- **"Грязные руки":** Пальцы атлетов покрыты магнезией (белый порошок), что снижает точность касаний и чувствительность сенсора.
- **Освещение:** Ангарное освещение с высокой контрастностью и бликами.
- **Спешка:** Запись часто происходит "на ходу" между подходами к стене.

## 2. Дизайн-принципы

| # | Принцип | Что это значит для макета |
|---|---------|---------------------------|
| P1 | **Магнезия-friendly** | Минимальный размер тач-зон **48x48 pt**. Крупные кнопки во всю ширину экрана. |
| P2 | **Smart UI (Widget over History)** | Полноценный раздел истории в MVP заменён на **динамический виджет** активной брони над списком тренировок. |
| P3 | **Высокая читаемость** | Контрастность по стандарту **WCAG AA**. Крупные заголовки и жирные шрифты для ключевых данных (время, места). |
| P4 | **Тактильный фидбек** | Использование **виброотклика (Haptic)** при успешных действиях (бронь, отмена) для подтверждения контакта. |
| P5 | **Спортивный Tone of Voice** | Дружелюбный, бодрый тон, общение на «ты». Мотивация вместо сухих системных сообщений. |

## 3. Каркас и Навигация

### 3.1 Структура Таб-бара (Нижнее меню)
Для минимизации затрат на разработку (BR-7) в MVP используется упрощенная навигация:
1. **Тренировки (Главная):** Список всех слотов + Smart-виджет текущей записи.
2. **Профиль:** Личные данные (в т.ч. размер обуви) и настройки аккаунта.

### 3.2 Глобальные элементы
- **Offline Banner:** Появляется под хедером при потере сети. Текст: «Нет соединения. Показываем данные из кэша».
- **System Loader:** Использование скелетонов (Shimmer effect) вместо блокирующих экран спиннеров.
