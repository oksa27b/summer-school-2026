# Описание домена и ограничения

> Этап 1. Артефакт 1 проекта: согласованное описание домена + список ограничений. 
> Обновлено по результатам ревью: исправлена логика лимитов и уточнена групповая запись.

## Описание домена

**Бизнес.** Скалодром «Вертикаль» проводит групповые тренировки по скалолазанию. Занятие длится 1.5 часа. Форматы: Болдеринг (новичковый, лимит 8 чел) и Высокая стена (опытные, лимит 16 чел). 

**Процесс.** Владелец формирует расписание на неделю. Клуб располагает **12 комплектами снаряжения** (скальники + страховочная система). Сейчас запись ведётся вручную, что приводит к овербукингу и путанице.

**Цель клиентского приложения.** Самостоятельная онлайн-запись клиентов (просмотр слотов, бронирование, выбор снаряжения, отмена). Бэкенд — существующая инфраструктура (Black-box), приложение потребляет данные через API.

**Ключевые объекты домена:**
- **Тренировка / слот** — сеанс в расписании: время, формат, инструктор, доступные места.
- **Формат / Зона** — Болдеринг (лимит 8) или Высокая стена (лимит 16). Ограничивает физическое кол-во людей.
- **Снаряжение** — инвентарь (12 комплектов). Ограничивает только тех, кому нужен прокат.
- **Бронь** — запись на 1–3 человек с индивидуальным выбором снаряжения для каждого.
- **Клиент** — пользователь, может хранить в профиле размер обуви для ускорения выдачи проката.

## Ограничения

**Бизнес-ограничения**

- **R-001 (Раздельные лимиты):** Система контролирует два независимых ресурса. Запись возможна, если:
  1. `Запрашиваемые места <= (Лимит формата - Занято мест)`.
  2. `Запрашиваемый прокат <= (12 - Выдано комплектов)`.
- **R-002 (Групповая бронь):** Один клиент может забронировать до 3-х мест. Вариант «Своё / Прокат» выбирается **индивидуально для каждого** из 3-х участников.
- **R-003 (Политика отмен):** Ранняя отмена (≥2 ч) — место и инвентарь возвращаются в фонд. Поздняя (<2 ч) — место остается занятым (доска/место простаивает), инвентарь в цифровом виде не возвращается в фонд до конца слота, штрафа нет.
- **R-004 (Оплата):** Только офлайн (наличные / перевод в зале). Приложение показывает итоговую стоимость.
- **R-027 (Горизонт):** Список слотов — 7 дней. При отсутствии мест на все дни показывается **Full State** («На ближайшие дни всё забронировано»).

**Технические / платформенные**

- **Mobile-first:** Интерфейс для работы **руками в магнезии** (крупные тач-зоны ≥ 44pt, высокий контраст).
- **Black-box API:** Инфраструктура и SMS-шлюз уже реализованы на стороне бэкенда.
- **Source of Truth:** Бэкенд — единственный источник расчета лимитов и дедлайнов отмены.

**Границы скоупа (MVP vs Phase 2)**

| В скоупе (MVP) | Вне скоупа (Phase 2) |
| :-- | :-- |
| Регистрация по телефону (SMS OTP) | Сбор оценок и текстовых отзывов |
| Просмотр тренировок и фильтрация | Онлайн-оплата (эквайринг) |
| Запись (1-3 чел) с выбором снаряжения | Лист ожидания (Waitlist) |
| Отображение флага «Топ-инструктор» | Справочник инструкторов с BIO |

## Глоссарий

- **Прокат:** Аренда снаряжения (скальники + система). Своё снаряжение не расходует лимит из 12 штук.
- **Топ-инструктор:** Справочный статус тренера (иконка «Звезда» в карточке слота).
- **Поздняя отмена:** Отмена менее чем за 2 часа. Место считается занятым, чтобы не нарушать план тренировки.
