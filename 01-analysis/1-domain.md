# Этап 1. Описание домена и ограничения

## 1. Описание предметной области
Скалодром «Вертикаль» предоставляет услуги групповых тренировок. 
- **Продукт:** Слот в расписании (время, формат, инструктор).
- **Ресурсы:** Инструкторское время (лимит 8/16 человек) и физический инвентарь (12 комплектов снаряжения).

## 2. Роли
- **Клиент (в скоупе):** Пользователь мобильного приложения. Цель: найти слот, записаться, управлять своими бронями.
- **Инструктор/Админ (вне скоупа):** Работают через существующую инфраструктуру.

## 3. Бизнес-ограничения
- **R-001 (Лимиты):** Максимальное число мест в записи — 3.
- **R-002 (Снаряжение):** Если `free_equipment_sets = 0`, запись доступна только при выборе опции «Своё снаряжение».
- **R-003 (Время):** Правило 2 часов для отмены (UTC серверное время).
- **R-004 (Горизонт):** Список слотов по умолчанию — 7 дней.

## 4. Технические ограничения
- **R-005 (Black Box):** Бэкенд уже реализован. Мы проектируем только контракт взаимодействия (API).
- **R-006 (Mobile-first):** Интерфейс должен учитывать использование в зале (крупные элементы, высокий контраст).
